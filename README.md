# Optimizing-Congestion-in-High-Load-Singapore-MRT-Stations
A research proposal exploring mathematically grounded operational strategies to reduce congestion in high-load Singapore MRT stations using passenger-flow and queuing models.

## Project Overview
This project investigates congestion in major Singapore MRT stations and evaluates possible operational interventions such as:
- Time-based escalator direction control
- Crowd separation using physical barriers

The study combines:
- Empirical passenger-load evidence
- Queuing theory
- Crowd dynamics modelling
- Performance indicators for congestion evaluation

## Repository Structure
- `docs/` original proposal documents
- `proposal/` markdown version of the proposal
- `assets/` diagrams and figures
- `data/` data notes and sources

## Research Focus
The proposal focuses on high-load MRT stations in Singapore and aims to bridge theoretical modelling with practical station operations.
